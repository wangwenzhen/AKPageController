# AKPageController
市面流行的上下区域相互滚动



使用方法。

1.将AKPageController 拖入工程

2.载体vc参考AKDemoVC

3.底部滚动参考AKCommenVC

4.手动拖入BeautyKit 中的工具类 即可。  [可以移除将使用的工具类替换自己的工具类]

5.集成pod

pod 'YYCategories','1.0.4'

pod 'ReactiveObjC','3.1.1'

pod 'Masonry','1.1.0'

pod 'SDWebImage','4.4.6'     

pod 'SDWebImage/GIF', '4.4.6'
